Audio Downloader

A simple GUI application for downloading audio from various sources including YouTube.

Download

You can download the latest version of Audio Downloader:

Go to GitHub Actions Direct Download

Note: For GitHub Actions, go to the latest successful workflow run, scroll down to the "Artifacts" section, and download "AudioDownloader-Windows-ZIP".

Features

  • Download audio from YouTube videos
  • Download audio from direct URLs
  • Select custom save location
  • Real-time download progress updates
  • Support for multiple download methods (pytube, youtube-dl, yt-dlp)

Requirements

This application requires Windows 10/11.

Installation

  1. Download the ZIP file using one of the download buttons above
  2. Extract the contents
  3. Run AudioDownloader.exe
